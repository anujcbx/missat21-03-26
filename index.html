λ Foundation

What is LAMBDA?

LAMBDA is a Google Sheets function that lets you define your own custom function — with named inputs and your own formula logic — saved permanently under a name you choose.

Without LAMBDA
❌ Repeated in every row
You type this in D2, D3, D4… 500 times
=IF((C2-B2)/C2*100>30,"High", IF((C2-B2)/C2*100>15,"Med","Low"))

Change threshold? Fix 500 cells manually.

With LAMBDA
✅ Define once, call anywhere
Define in Data → Named Functions (once)
=LAMBDA(sale, cost, LET(m,(sale-cost)/sale*100, IFS(m>30,"High",m>15,"Med",TRUE,"Low")) )
Call it from any cell
=MARGIN_BAND(C2, B2)
💡 Once saved as a Named Function, LAMBDA autocompletes in the formula bar — just like =SUM or =IF. Your whole team can use it without knowing the formula inside.

LAMBDA + Helper Functions

LET inside LAMBDA
Store interim values cleanly

LET creates named variables inside your formula — calculate once, reuse anywhere inside the LAMBDA body.

=LAMBDA(basic, LET( hra, basic*0.40, pf, basic*0.12, basic + hra - pf - 200 ) )
MAP — Apply to Whole Column
One formula for 1000 rows

MAP applies your Named Function to every row in a range automatically — no dragging, no copying.

Apply NET_SALARY to all of column B =MAP(B2:B500, NET_SALARY) Apply to two columns (2 arguments) =MAP(C2:C500, D2:D500, MARGIN_BAND)
λ Anatomy

Formula Anatomy

Break down every part of a LAMBDA formula so you can read, write, and debug any LAMBDA with confidence.

Full LAMBDA Structure
=LAMBDA( param1, param2, formula_body )
LAMBDA(
The wrapper — tells Sheets this is a custom function definition.
param1, param2
Your input names — you invent these. Use descriptive names: sale_price, basic_salary, dispatch_date.
formula_body
The last argument — always the formula. Use your param names here instead of cell references like B2.
When called
When you type =MY_FUNC(A2, B2), A2 slots into param1 and B2 into param2 throughout the formula.

GST Calculator — Full Breakdown

Step 1 — The raw formula (in a single cell)
=B2 + (B2 * 18 / 100)
Step 2 — Wrap in LAMBDA with named params
=LAMBDA(base_amount, gst_rate, base_amount + (base_amount * gst_rate / 100) )
Step 3 — Save as Named Function "GST_TOTAL" → Use it
=GST_TOTAL(B2, 18) → ₹11,800 =GST_TOTAL(B3, 5) → ₹2,100 =GST_TOTAL(B4, 28) → ₹6,400 =MAP(B2:B1000, C2:C1000, GST_TOTAL) → Apply to all rows
λ Patterns

Core Patterns

Six reusable LAMBDA patterns that cover 90% of all business formula needs. Learn these and you can build any tool.

Pattern 1 — Simple IF
=LAMBDA(x,IF(x>0,"Profit","Loss"))
Pattern 2 — IFS Multi-Band
=LAMBDA(score,IFS(score>=90,"A",score>=75,"B",score>=60,"C",TRUE,"F"))
Pattern 3 — LET + IFS
=LAMBDA(a,b,LET(pct,a/b*100,IFS(pct>=90,"✅",pct>=75,"⚠️",TRUE,"🔴")))
Pattern 4 — Date Logic
=LAMBDA(dt,sla,IF(TODAY()-dt>sla,"DELAYED","On Time"))
Pattern 5 — Arithmetic + Text
=LAMBDA(s,c,LET(m,ROUND((s-c)/s*100,1),m&"%"))
Pattern 6 — IFERROR Safety
=LAMBDA(a,b,IFERROR(IF(b=0,"—",a/b*100),"Error"))

When to Use MAP vs ARRAYFORMULA

MAP — for Named Functions

Use MAP when applying a LAMBDA Named Function to a range. It passes each row into your function correctly.

=MAP(B2:B500, NET_SALARY) =MAP(C2:C500,D2:D500,MARGIN_BAND)
ARRAYFORMULA — for inline logic

Use ARRAYFORMULA for simple inline formulas without a Named Function — like IF or arithmetic on a range.

=ARRAYFORMULA(B2:B500*1.18) =ARRAYFORMULA(IF(A2:A500<>"",B2:B500*C2:C500,""))
λ How To Create

How to Create a Named Function

Step-by-step guide to saving any LAMBDA as a permanent Named Function in Google Sheets — takes under 2 minutes.

1

Open your Google Sheet → Click "Data" in the menu

In the top navigation bar, click Data → scroll to the bottom → click Named Functions.

2

Click "+ Add new function"

A dialog panel opens on the right side. This is where you define your custom LAMBDA function.

3

Enter a Function Name (CAPS, no spaces)

Example: NET_SALARY, GST_TOTAL, DELAY_FLAG. This becomes the formula name you'll type in cells.

4

Add Argument Placeholders

Click + Add argument placeholder for each input. Name them clearly: basic, sale_price, dispatch_date. These are the parameter names used inside your formula.

5

Write the Formula using argument names

In the formula box, write your LAMBDA formula using argument names instead of cell references. Example: use basic instead of B2.

6

Click Create → Use it like any built-in function

Type =NET_SALARY(B2) in any cell. Autocomplete works. Any team member can use it. Change the logic in Named Functions → all cells update instantly.

⚡ Pro tip: Named Functions are sheet-specific. To reuse them across projects, keep a "Master Template" sheet with all your Named Functions defined — copy the sheet to start new projects.

🔄 App Build Workflow

How to Build a Sheet App

A 5-step process from idea to deployed client tool — using only AI Studio prompts and Google Sheets.

1

Define Your Tool (Answer 4 questions)

Who uses it? What do they enter? What should auto-calculate? What decision does the output drive? Write this in 2–3 sentences — it becomes your AI prompt.

2

Use the Dept/Industry Prompt from this section

Find your department or industry tab. Copy the prompt. Paste into AI Studio, ChatGPT, or Claude. The AI returns a complete app blueprint.

3

Build the 3-Tab Sheet Structure

Tab 1 — Data: Input columns + auto-calculated output columns. Tab 2 — Dashboard: COUNTIF/FILTER summaries for management. Tab 3 — Config: Rates, thresholds, parameters — change here, everything updates.

4

Add Conditional Formatting for Visual Alerts

Format → Conditional formatting → Custom formula. Set red rows for critical alerts, amber for warnings, green for OK. The sheet becomes a visual dashboard — scannable in seconds.

5

Deploy: Lock inputs, Hide formulas, Share link

Protect input ranges so formulas can't break. Hide formula columns. Share the Dashboard tab with your client as View Only — they get a live, professional report.

⏱️ Time estimate: A basic utility tool takes 1–2 hours from prompt to live sheet. A full multi-department MIS system takes a day. Either way — no code, no developers.

Deploy as a Client Tool

Turn your AI-built Sheet app into a professional, shareable client tool — without any extra software or hosting costs.

1

Lock Input Ranges

Data → Protect sheets and ranges → Lock all columns except user input cells. Users can enter data but can't break your formulas. Show a clear message: "This cell is protected — contact [your name] to modify."

2

Hide Formula & Config Columns/Tabs

Right-click column headers → Hide columns for any helper/formula columns. Right-click the Config tab → Hide sheet. The tool looks clean — users only see what they need.

3

Share Dashboard Tab as View Only

Share the Google Sheet → select "Viewer" access. Or better: right-click the Dashboard tab → Copy to → New spreadsheet. Share only the dashboard file — clients see live data without accessing the raw data tab.

4

Publish as a Web App (Zero Login Required)

File → Share → Publish to web → Entire document or specific sheet → Embed link. Anyone with the link can view your live dashboard in a browser without needing a Google account — perfect for client-facing reports.

5

Upgrade to Looker Studio for Premium Look
